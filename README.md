# projet1-pratiquepro
Audit et Plan d’Action pour la Cybersécurité de E-Commerce Express

## Description
Ce dépôt a été créé dans le cadre du Projet 1 du cours INF1753.  
Il sert exclusivement à démontrer l’utilisation de GitHub comme outil de collaboration et de versionnement, tel qu’exigé dans les consignes du professeur ( Dr. N’dah Daniel Yapi )

## Contenu
- Un fichier test (PDF ) démontrant l’ajout de fichiers dans un dépôt GitHub.
Ce fichier contient les énoncées du projet 

## Objectif du dépôt
- Illustrer la création d’un dépôt GitHub public.
- Montrer comment ajouter un fichier et valider un commit.
- Fournir le lien à inclure dans le rapport final dans la section *Annexe C*.
